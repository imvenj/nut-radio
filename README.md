# nut-radio

一个为试用 [nutui](https://github.com/jdf2e/nutui) 而开始的小项目，主要功能是业余无线电考试学习和自测。

## 编译使用

```bash
npm install # 安装依赖
npm run serve # 启动服务
npm run build # 编译部署代码
```

## 题库数据

题库数据来自[中国无线电协会业余无线电分会网站](http://www.crac.org.cn/News/Detail?ID=1862)，仅供爱好者自学使用，请勿用于商业目的。
